# Authentication System Styling

A fully styled, responsive authentication system built with Bootstrap 5 and custom CSS. This project transforms a plain HTML authentication system into a modern, professional web application.

## 📋 Project Overview

This project is part of the Fullstack Java Development course at CampusPe. It extends a previous HTML-only authentication system by applying professional styling using Bootstrap 5 and custom CSS.

**Features:**
- 5 fully styled authentication pages
- Responsive design for all devices
- Bootstrap 5 components (cards, forms, navbar)
- Custom CSS with professional color scheme and hover effects
- Mobile-first responsive layout
- Professional dashboard with statistics cards

## 📄 Pages Overview

| Page | Description | Key Elements |
|------|-------------|--------------|
| **Login Page** | User authentication entry point | Email field, password field, login button, forgot password link, create account link |
| **Registration Page** | New user account creation | Full name, email, password fields, register button, back to login link |
| **Forgot Password Page** | Password recovery | Email input field, send link button |
| **Reset Password Page** | Password reset | New password field, update button |
| **Dashboard Page** | User dashboard | Navbar navigation, welcome message, user email, statistics cards (Users, Orders, Revenue, Reports), logout button |
